# Matchify Mobile (Sportify Hub)

This is the Expo-based mobile application for Matchify.

## ✨ Features
- **Dark Theme**: Fully optimized dark mode.
- **Glassmorphism**: Premium UI with `BlurView` cards.
- **Mock Data**: Decoupled from backend for testing.
- **Full Navigation**: Home, Explore, Play, Activity, and Profile.

## 🏃 How to Run
1. Ensure you have the [Expo Go](https://expo.dev/go) app on your device.
2. `npm install`
3. `npx expo start`
4. Scan the QR code with your camera (iOS) or Expo Go (Android).

## 🛠️ Tech Stack
- React Native / Expo
- React Navigation (Native Stack + Bottom Tabs)
- Lucide React Native (Icons)
- TypeScript
